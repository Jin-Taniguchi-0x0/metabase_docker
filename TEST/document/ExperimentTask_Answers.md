# タスクシナリオ解答集

## 1. オリンピックデータ (`ExperimentTask_Hypothesis_Olympics.md`)

> 「歴史的に見て、**[ A ]** はある2つの競技で熾烈なトップ争いをしていますが、特に **[ B ]** においては、**[ C ]** に **300個を超えるメダル** を獲得しており、これが単一競技における10年間の最多記録（黄金期）のようです。
> また、開催地（City）の影響も無視できません。この国は、自国開催となった **[ D ]** 大会において、チーム競技のメダルを含め **700個近く** という驚異的な数のメダルを獲得しています」

*   **[ A ]: USA** (アメリカ合衆国)
    *   候補の `USA` と `URS` のうち、以下の条件を満たすのはUSAのみ。
*   **[ B ]: Swimming** (水泳)
*   **[ C ]: 2000年代**
    *   データ: USAは2000年代にSwimmingで **319個** のメダルを獲得。
    *   (参考: Athleticsの最高は1980年代の237個)
*   **[ D ]: Los Angeles** (ロサンゼルス)
    *   データ: USAは1984年ロサンゼルス大会で **689個** (約700個) のメダルを獲得。
    *   (参考: St. Louis 1904は394個, Atlanta 1996は259個)

## 2. ワインレビューデータ (`ExperimentTask_Hypothesis_Wine.md`)

> 「一般的に評価点（Points）が高いワインは価格（Price）も高いが、**[ A ]** の **[ B ]** に関しては、比較的低価格でありながら **[ C ]** の高評価を獲得しているものが多く、『コスパ最強』のカテゴリーと言えるのではないか。
> また、著名なレビュワーである **Kerin O’Keefe** 氏は、特に **[ D ]** のワインに精通しているようだが、**価格が $60 を超えるハイエンドワイン** については、他のレビュワーと比べて **[ E ]** という特徴があるようだ」

*   **[ A ]: Portugal** (ポルトガル) ※またはデータ分析に基づく高コスパ国
    *   データ分析例: 平均価格が低く、平均点が高い国として `Portugal` や `Austria` が挙がることが多い。
*   **[ B ]: Portuguese Red** (または特定の土着品種)
*   **[ C ]: 90点以上** (または88点以上)
*   **[ D ]: Italy** (イタリア)
    *   Kerin O’Keefe氏のレビューの大半はイタリアワイン。
*   **[ E ]: 評価が厳しい / 平均点が低い**
    *   データ分析結果: イタリアのハイエンドワイン(>$60)において、Kerin O’Keefe氏の平均点は他のレビュワーよりも低い傾向がある (または分散が小さい)。

## 3. 超大国の激突 (`ExperimentTask_Hypothesis_Superpower.md`)

> 「単純な国別ランキングではアメリカが常に上位にいるように見える。しかし、『旧ソ連・ロシア圏（URS + RUS + EUN）』を合算して比較すると、**[ A ]** から **[ B ]** にかけては、実はアメリカを凌駕するメダルを獲得していたようだ。
> 特に **[ C ]** においてはその差が顕著で、この期間、アメリカは『旧ソ連・ロシア圏』の牙城をほとんど崩せていない。
> さらに興味深いのは季節による違いだ。通算メダル数ではアメリカが勝っているが、**[ D ]** オリンピックに限って言えば、実は『旧ソ連・ロシア圏』の方が **[ E ]** も多くのメダルを獲得しており、真の王者と言えるかもしれない」

*   **[ A ]: 1950年代**
*   **[ B ]: 1980年代** (または1990年代初頭)
    *   データ: 1956年〜1992年(EUN)の間、多くの大会で旧ソ連圏がメダル総数でUSAを上回る。
*   **[ C ]: Gymnastics** (体操)
    *   データ: この期間、旧ソ連圏は体操で圧倒的なメダル数を獲得。
*   **[ D ]: Winter** (冬季)
    *   データ: 冬季五輪の通算メダル数は、旧ソ連・ロシア圏 (URS+RUS+EUN) が USA を上回る。
*   **[ E ]: 100個以上** (約120個差)
    *   データ: URS/RUS/EUN (約759個) vs USA (約635個)

*   **[ A ]:** (データ分析に基づくターゲット層、例: `Men 18-24` など)
*   **[ B ]: Instagram** (またはデータに基づく最高ROIチャネル)
*   **[ C ]: Instagram** (またはデータに基づく最低コストチャネル)

## 5. UFOデータ (`ExperimentTask_Hypothesis_UFO.md`)

> 「UFOの目撃情報は **[ A ]** にかけて急増しており、特に **[ B ]** において『 **[ C ]** 』の形状をしたUFOが多く目撃されているようだ」

*   **[ A ]: 2000年代** (または1990年代後半〜)
    *   データ: 2000年代以降に目撃情報が激増。
*   **[ B ]: us** (アメリカ合衆国)
    *   データ: 圧倒的多数がUSでの目撃。
*   **[ C ]: light** (光)
    *   データ: 形状としては `light` が最も多く、次いで `triangle`, `circle`。
