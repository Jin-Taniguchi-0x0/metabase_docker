### オリンピックデータ タスクシナリオ

### **📌 プロジェクト概要**

スポーツアナリティクスチームより、近代オリンピックの歴史データ（`athlete_events`）を用いた分析依頼が届きました。
チームは、特定の国や競技における「黄金期」の存在や、開催地が結果に与える影響（ホームアドバンテージ）について議論していますが、確固たる証拠を持っていません。

あなたのミッションは、**インタラクティブなダッシュボードを作成し、データに基づいてチームの仮説を検証・特定すること**です。

### **💼 チームの仮説（検証対象）**

チームはデータから逆算された以下の仮説を持っていますが、**[ ]** の部分を伏せています。ダッシュボードを作成し、この空白をファクトで埋めてください。

> 「歴史的に見て、**[ A ] （国名）** はある2つの競技で熾烈なトップ争いをしていますが、特に **[ B ] （競技名）** においては、**[ C ] 年代** に **300個を超えるメダル** を獲得しており、これが単一競技における10年間の最多記録（黄金期）のようです。
> また、開催地（City）の影響も無視できません。この国は、自国開催となった **[ D ] （都市名）** 大会において、チーム競技のメダルを含め **700個近く** という驚異的な数のメダルを獲得しています」
>
> **※ [ A ] の候補: USA, URS**

### **📃 納品物（作成するダッシュボードの要件）**

この仮説を検証するために、以下の分析視点を網羅したダッシュボードを作成してください。

**1. 国別・競技別メダル獲得推移（[ A ], [ B ], [ C ] の特定）**

- **目的:** 特定の国が特定の競技でいつ最も多くのメダルを獲得したか（黄金期）を特定する。
- **ヒント:**
    - フィルタを使って、メダル獲得数上位の国（候補の2ヶ国）と主要競技（例: Athletics, Swimming, Gymnasticsなど）の組み合わせを探索してください。
    - 「10年間で300個以上」という条件に合致する [ B ] と [ C ] を見つけてください。

**2. 開催地とメダル獲得数の関係**

- **目的:** [ A ] の国が開催国となった都市（City）でのメダル獲得数を、他の大会と比較する。
- **ヒント:**
    - 開催都市（City）ごとのメダル獲得数を棒グラフなどで表示し、[ A ] の国の都市が上位（700個近く）に来ているか確認してください。

**3. アスリートの身体的特徴の変遷**

- **目的:** [ B ] の競技において、アスリートの身長（Height）や体重（Weight）が時代とともにどのように変化（大型化など）しているかを確認する。

**4. 追加分析：ホームアドバンテージの検証**

- **目的:** `City`（開催地）と `NOC`（国・地域）の関係に着目し、開催国が有利になる「ホームアドバンテージ」がどの程度存在するかを検証する。

### **🚀 実行手順**

1. 分析アプリケーションにログインし、データソース `athlete_events` に接続します。
2. **まず最初に「折れ線グラフ」を作成**し、横軸に `Year`、縦軸に `Count`（メダル数）を設定して、全体のメダル獲得数の推移を可視化してください。（これは必須のステップです）
3. 続けて、上記の分析視点に対応する他のグラフを作成し、ダッシュボードに配置します。
4. 完成したダッシュボードを操作し、仮説の **[ A ] 〜 [ D ]** に当てはまる具体的なファクトを特定してください。
5. ホームアドバンテージが実際に数値としてどれくらい現れているか（例：他大会の平均と比べて何倍かなど）を報告してください。
