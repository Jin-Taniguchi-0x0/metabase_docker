### ソーシャルメディア広告データ タスクシナリオ

### **📌 プロジェクト概要**

デジタルマーケティング部より、過去の広告キャンペーンデータ（`social_media_ads`）に基づいた予算配分の最適化案を求められています。
マーケティング部長は、ターゲット層ごとに最適なチャネルがあると考えていますが、費用対効果（ROI）と獲得コスト（CPA）のバランスについては確信を持てていません。

あなたのミッションは、**インタラクティブなダッシュボードを作成し、データに基づいて最適な広告戦略を導き出すこと**です。

### **💼 部長の仮説（検証対象）**

部長は以下の仮説を持っていますが、**[ ]** の部分をデータで裏付ける必要があります。

> 「[ A ] （ターゲット層）に対しては、[ B ] （チャネル名）が最も高いROI（投資対効果）を出しているようだ。しかし、獲得コスト（Acquisition_Cost）を抑えることを最優先するなら、むしろ [ C ] （チャネル名）の方が効率的かもしれない。キャンペーンの目的（Campaign_Goal）によっても使い分けが必要だろう」
> 

### **📃 納品物（作成するダッシュボードの要件）**

この仮説を検証するために、以下の分析視点を網羅したダッシュボードを作成してください。

**1. ターゲット層別・チャネル別ROI分析（[ A ], [ B ] の特定）**

- **目的:** どのターゲット層に対して、どのチャネルが最も高いROIを記録しているかを特定する。
- **ヒント:** ヒートマップや棒グラフで比較すると分かりやすいでしょう。

**2. 獲得コスト（Acquisition_Cost）の比較分析（[ C ] の特定）**

- **目的:** チャネルごとの獲得コストの分布や平均値を比較し、最もコスト効率の良いチャネルを見つける。
- **注意:** `Acquisition_Cost` はテキスト形式（例: "$50"）の場合があるため、数値として扱えるか確認が必要です。

**3. キャンペーン目的別のパフォーマンス分析**

- **目的:** `Campaign_Goal`（認知拡大、コンバージョンなど）ごとに、成功しているチャネルや戦略が異なるかを確認する。

### **🚀 実行手順**

1. 分析アプリケーションにログインし、データソース `social_media_ads` に接続します。
2. **まず最初に「ピボットテーブル」または「棒グラフ」を作成**し、チャネル別の主要指標（ROI, Clicksなど）を概観します。
3. 続けて、上記の分析視点に対応する他のグラフを作成し、ダッシュボードに配置します。
4. 完成したダッシュボードを操作し、仮説の **[ A ], [ B ], [ C ]** に当てはまる具体的なファクトを特定してください。
5. 予算配分の提案として、「ROI重視プラン」と「コスト抑制プラン」の2つのシナリオを裏付けるデータを提示してください。
