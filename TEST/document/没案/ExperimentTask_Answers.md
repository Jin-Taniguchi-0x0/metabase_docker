# タスクシナリオ解答集

## 1. オリンピックデータ (`ExperimentTask_Hypothesis_Olympics.md`)

> 「歴史的に見て、**[ A ]** はある 2 つの競技で熾烈なトップ争いをしていますが、特に **[ B ]** においては、**[ C ]** に **300 個を超えるメダル** を獲得しており、これが単一競技における 10 年間の最多記録（黄金期）のようです。
> また、開催地（City）の影響も無視できません。この国は、自国開催となった **[ D ]** 大会において、チーム競技のメダルを含め **700 個近く** という驚異的な数のメダルを獲得しています」

- **[ A ]: USA** (アメリカ合衆国)
  - 候補の `USA` と `URS` のうち、以下の条件を満たすのは USA のみ。
- **[ B ]: Swimming** (水泳)
- **[ C ]: 2000 年代**
  - データ: USA は 2000 年代に Swimming で **319 個** のメダルを獲得。
  - (参考: Athletics の最高は 1980 年代の 237 個)
- **[ D ]: Los Angeles** (ロサンゼルス)
  - データ: USA は 1984 年ロサンゼルス大会で **689 個** (約 700 個) のメダルを獲得。
  - (参考: St. Louis 1904 は 394 個, Atlanta 1996 は 259 個)

## 2. ワインレビューデータ (`ExperimentTask_Hypothesis_Wine.md`)

> 「弊社の基準では、Points が **80~85 点であれば『一般』**、**85~90 であれば『高品質』**、**90 以上であれば『最高品質』** と定義しています。
> 一般的に評価点（Points）が高いワインは価格（Price）も高い傾向にありますが、**この相関から外れた『コスパ最強』の生産国**が存在するはずです。
> 具体的には、**[ A ]** は平均価格が世界平均と比べて **$ [ B ]** も安く、中でも **[ C ]** は圧倒的なコストパフォーマンスを誇っていると推測しています。この仮説を検証するためのダッシュボードを作成し、該当する国と詳細を特定してください。」

- **[ A ]: Romania** (ルーマニア)
  - データ: 平均価格が約 **$15.2** と非常に安く、かつ平均 Points が **86.4** (高品質基準の 85 以上) であるため。
- **[ B ]: 20.1** (約 $20)
  - 世界平均 ($35.4) と Romania 平均 ($15.2) の差。
- **[ C ]: Moscato / Pinot Grigio** (または Pinot Noir など)
  - Points/Price のスコアが最も高い品種。Moscato はスコア 11.9, Pinot Grigio は 11.3 など。
  - (Pinot Noir や Cabernet Sauvignon も高コスパで正解としても良い)

※ **自由課題 (レビュワー分析)** について

- **正解の方向性:** Romania のメインレビュワーは **Jeff Jenssen** (66 件) と **Anna Lee C. Iijima** (41 件) です。
- **分析例:** 彼らの Romania に対する平均評価点（約 86 点台）と、彼らの全体的な平均評価点（約 88 点台）を比較すると、Romania に対しては「やや厳しめ」か「標準的（85 点以上）」な評価をしていることが分かります。このような視点が含まれていれば正解です。

## 3. 超大国の激突 (`ExperimentTask_Hypothesis_Superpower.md`)

> 「単純な国別ランキングではアメリカが常に上位にいるように見える。しかし、『旧ソ連・ロシア圏（URS + RUS + EUN）』を合算して比較すると、**[ A ]** から **[ B ]** にかけては、実はアメリカを凌駕するメダルを獲得していたようだ。
> 特に **[ C ]** においてはその差が顕著で、この期間、アメリカは『旧ソ連・ロシア圏』の牙城をほとんど崩せていない。
> さらに興味深いのは季節による違いだ。通算メダル数ではアメリカが勝っているが、**[ D ]** オリンピックに限って言えば、実は『旧ソ連・ロシア圏』の方が **[ E ]** も多くのメダルを獲得しており、真の王者と言えるかもしれない」

- **[ A ]: 1950 年代**
- **[ B ]: 1980 年代** (または 1990 年代初頭)
  - データ: 1956 年〜1992 年(EUN)の間、多くの大会で旧ソ連圏がメダル総数で USA を上回る。
- **[ C ]: Gymnastics** (体操)
  - データ: この期間、旧ソ連圏は体操で圧倒的なメダル数を獲得。
- **[ D ]: Winter** (冬季)
  - データ: 冬季五輪の通算メダル数は、旧ソ連・ロシア圏 (URS+RUS+EUN) が USA を上回る。
- **[ E ]: 100 個以上** (約 120 個差)
  - データ: URS/RUS/EUN (約 759 個) vs USA (約 635 個)

## 4. ソーシャルメディア広告データ (`ExperimentTask_Hypothesis_SocialMedia.md`)

- **[ A ]:** (データ分析に基づくターゲット層、例: `Men 18-24` など)
- **[ B ]: Instagram** (またはデータに基づく最高 ROI チャネル)
- **[ C ]: Instagram** (またはデータに基づく最低コストチャネル)
  - データ分析結果: Instagram は Acquisition Cost が最も低い傾向にある ($7726 vs $7774 Twitter)。

## 5. UFO データ (`ExperimentTask_Hypothesis_UFO.md`)

> 「UFO の目撃情報は **[ A ]** にかけて急増しており、特に **[ B ]** において『 **[ C ]** 』の形状をした UFO が多く目撃されているようだ」

- **[ A ]: 2000 年代** (または 1990 年代後半〜)
  - データ: 2000 年代以降に目撃情報が激増。
- **[ B ]: us** (アメリカ合衆国)
  - データ: 圧倒的多数が US での目撃。
- **[ C ]: light** (光)
  - データ: 形状としては `light` が最も多く、次いで `triangle`, `circle`。
